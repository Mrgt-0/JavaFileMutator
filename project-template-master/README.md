# Мутационный фаззер для статического анализатора.

Инструмент для генерации мутантов Java-кода с использованием библиотеки Spoon. Позволяет создавать модифицированные версии Java-файлов для тестирования качества тестов и статических анализаторов.

## Инструменты
- Gradle 9.4.0
- Java 25
- JUnit + AssertJ
- Spoon 11.2.1

## Функции

- Замена операторов сравнения
- Добавление скобок вокруг операндов
- Усложнение операндов арифметическими операциями
- Добавление приведения типов
- Изменение контекста выражений
- Замена типов переменных

## Сборка

```bash
./gradlew build
```

## Запуск

```bash
./gradlew run
```

Точка входа - `me.template.Main`.

## Тесты

```bash
./gradlew test
```
